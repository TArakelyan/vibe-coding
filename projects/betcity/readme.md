# BETCITY Widget - Количество команд АПЛ в 1/4 ЛЧ

## Описание
Виджет для отображения коэффициентов на количество команд АПЛ в 1/4 финала Лиги Чемпионов.

## Варианты ставок
1. **Больше 2,5** - коэффициент 2.55
2. **Меньше 2,5** - коэффициент 1.45

## Особенности
- Все ссылки ведут на https://s.betcity.ru/aed
- Адаптивный дизайн для десктопа и мобильных устройств
- Пустое пространство вокруг виджета (padding: 20px)
- Темная тема поддерживается автоматически
- Кликабельные коэффициенты

## Размеры виджета

### Десктопная версия
- Ширина виджета: 730px (максимальная)
- Высота виджета: ~140px (автоматически подстраивается)

### Мобильная версия (до 768px)
- Ширина: 100% (адаптивная)
- Высота виджета: ~140px (автоматически подстраивается)

### Код для вставки
```html
<p><iframe data-height-desktop="150" data-height-mobile="150" data-height-tablet="150" src="ССЫЛКА_НА_DUMPSTER"></iframe></p>
```

Высота автоматически корректируется через resize.js для точного отображения без обрезки.

## Файлы
- `index.html` - основная структура виджета
- `styles.css` - стили виджета
- `main.js` - логика виджета
- `bundes-teams-data.js` - данные команд и коэффициентов
- `theme-detector.js` - определение темной темы
- `resize.js` - скрипт для изменения размера iframe
