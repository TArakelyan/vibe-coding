# Bundes Predictor

## Описание
Интерактивный виджет для отображения топ-3 претендентов Бундеслиги с коэффициентами от БЕТСИТИ. Виджет выполнен в стиле калькулятора с адаптивным дизайном и автоматическим определением темы.

## Особенности
- **Компактный дизайн**: фиксированная высота 200px на десктопе для встраивания в посты
- **Топ-3 команды**: Бавария, Байер, Боруссия Дортмунд
- **Коэффициенты**: отображение актуальных коэффициентов от БЕТСИТИ
- **Кликабельность**: весь виджет и коэффициенты ведут на БЕТСИТИ
- **Локальные логотипы**: использование локальных изображений команд
- **Автоматическая тема**: определение светлой/темной темы родительской страницы
- **Адаптивность**: корректное отображение на всех устройствах

## Команды
1. **Бавария** (Мюнхен) - коэффициент 1.25
2. **Байер** (Леверкузен) - коэффициент 10.00
3. **Боруссия Д** (Дортмунд) - коэффициент 10.00

## Технические особенности
- **Размеры**: 730px × 200px на десктопе
- **Автоматический resize**: для корректного отображения в iframe
- **Синий дизайн**: использование фирменного цвета БЕТСИТИ (#0093FA)
- **Шрифт**: TT Neoris для современного вида
- **Переходы**: все элементы ведут на https://s.betcity.ru/7cy

## Файлы проекта
- `index.html` - основная разметка
- `bundes-teams-data.js` - данные команд Бундеслиги
- `main.js` - логика работы виджета
- `styles.css` - стили в стиле калькулятора
- `resize.js` - автоматическое изменение размера iframe
- `theme-detector.js` - определение темы родительской страницы
- `bayern-logo.jpeg`, `bayer-logo.jpeg`, `dortmund-logo.jpeg` - логотипы команд
