# BETCITY Widget - Выход в 1/4 финала

## Описание
Виджет для отображения коэффициентов на выход в 1/4 финала Лиги Чемпионов.

## Команды
1. **Реал** - коэффициент 3.20
2. **Ман Сити** - коэффициент 1.36

## Особенности
- Все ссылки ведут на https://s.betcity.ru/a9x
- Адаптивный дизайн для десктопа и мобильных устройств
- Пустое пространство вокруг виджета (padding: 20px)
- Темная тема поддерживается автоматически
- Кликабельные коэффициенты

## Размеры виджета

### Десктопная версия
- Ширина виджета: 730px
- Общая ширина с отступами: 770px (730px + 20px * 2)
- Высота виджета: ~140px
- Общая высота с отступами: ~180px

### Мобильная версия (до 768px)
- Ширина: 100% - 40px (с учетом padding)
- Высота виджета: ~140px
- Общая высота с отступами: ~180px

## Файлы
- `index.html` - основная структура виджета
- `styles.css` - стили виджета
- `main.js` - логика виджета
- `bundes-teams-data.js` - данные команд и коэффициентов
- `theme-detector.js` - определение темной темы
- `resize.js` - скрипт для изменения размера iframe
