# Match Predictor

## Описание

Предиктор матчей Армянской Премьер-лиги: пользователь прогнозирует счёт, может поставить бустер ×2 на один матч в туре. Интерфейс в стилистике биржи прогнозов — чистый UI, полная ширина.

## Возможности

- 8 туров АПЛ (48 матчей), логотипы клубов
- Карточка матча: логотипы команд, поля счёта, кнопка сохранения и квадратный бустер
- Бустер ×2 — только на один матч в рамках тура
- Регистрация и вход по email (опционально телефон)
- Сброс пароля: по email (Firebase) или email/телефон (локальный режим)
- Переключатель языков: RU / HY / EN
- Таблица игроков с очками и фильтром по туру
- Автозакрытие прогнозов после kickoff (когда даты будут заданы)
- Система очков: исход, голы хозяев/гостей, разница, бонус за точный счёт

## Очки

| Прогноз | Баллы |
| --- | --- |
| Correct Outcome | +10 |
| Correct Goals (Home) | +5 |
| Correct Goals (Away) | +5 |
| Correct Goal Difference | +5 |
| Correct Score Bonus | +5 |

Бустер умножает итог матча на 2. Результаты матчей (`result` в данных) появятся позже — по ним считается рейтинг.

## Деплой

```bash
node build-standalone.mjs
```

Деплоить `index.html`. Исходники — `index.source.html` и отдельные файлы.

## Авторизация

По умолчанию — локальный режим (данные в браузере). Для облака включите Firebase в `data/config.js`.
