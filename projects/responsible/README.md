# Ответственная игра

## Описание

Компактный виджет об ответственной игре в обновлённом визуальном стиле: верхний блок с знаком «стоп» у заголовка и крупной типографикой, секции разделены тонкими линиями, карточки рекомендаций и встроенный тест.

## Функции

- **Шапка** без ярлыка: эмодзи знака «стоп» 🛑 у заголовка, усиленный заголовок; вводный текст в рамке с зелёной полосой (фон как у страницы)
- **Разделители** — тонкая линия `#d4d4d4` под каждой основной секцией
- **Визуал** в стиле editorial/sports: контрастная типографика и строгие рамки
- **Блок рекомендаций** с нумерованными карточками и акцентом на безопасную игру
- **Кнопка «НАЧАТЬ»** в верхней части для мгновенного запуска теста
- **Интерактивный тест в лендинге** (без поп-апа): фиксированная высота карточки вопроса, длинный текст автоматически уменьшается по размеру; после ответа — переход к следующему вопросу
- **Блоки помощи** со ссылками на Госуслуги и Единый ЦУПИС
- **Экстренная линия помощи** с кликабельным телефоном `8 800 600-31-39`

## Тест на игроманию

Тест состоит из 10 вопросов с 2 вариантами ответов:
- Да
- Нет

### Интерпретация результатов:

- **4 и более ответов «Да»** — есть признаки проблемного игрового поведения, рекомендуется обратиться за консультацией
- **Менее 4 ответов «Да»** — выраженных признаков проблемы не выявлено, важно продолжать ответственную игру

## Структура проекта

```
responsible/
├── index.html          # Главная страница (CSS и основные скрипты с dumpster CDN)
├── styles.css          # Исходные стили (дублируются на CDN)
├── script.js           # Исходный JS теста (дублируется на CDN)
├── theme-detector.js   # Исходник темы (дублируется на CDN)
├── resize.js           # Исходник resize iframe (дублируется на CDN)
├── data/
│   └── questions.js    # Вопросы теста (локально, подключается из ./data/)
└── README.md          # Документация
```

Подключения в `index.html`: [styles](https://dumpster.cdn.sports.ru/d/86/c56b79cdd2fdf1ecd7ea965f44de8.css), [script](https://dumpster.cdn.sports.ru/5/6e/eeda98dc1f611b575126bbd96a999.js), [theme](https://dumpster.cdn.sports.ru/d/75/7e43cdef780c8a04d791ec2026a92.js), [resize](https://dumpster.cdn.sports.ru/d/8f/095834ad31af26adbe8b0fc1fc09c.js).

### Выкладка на dumpster (один HTML)

- После правок **перезалейте** на CDN обновлённые `styles.css`, `theme-detector.js`, `script.js` и при необходимости `resize.js` — иначе в посте останется старая логика.
- **`theme-detector.js`** больше не задаёт тёмный градиент и светлый `color` на `body` при системной тёмной теме (из‑за этого текст на светлых карточках становился нечитаемым).
- Если рядом с HTML **нет** `./data/questions.js`, в `index.html` срабатывает **встроенный fallback** с теми же вопросами; при изменении `data/questions.js` синхронизируйте массив во встроенном скрипте.

## Дизайн

Проект использует адаптивный виджетный дизайн с максимальной шириной `660px`.

### Цветовая схема
- **Основной акцент**: `#00c78b` / `#43af8f`
- **Темный текст**: `#000` / `#111`
- **Фон секций**: светло-серый `#f5f5f5`

### Типографика
- **Заголовки**: Sports / Neoris
- **Текст**: Neoris / Roboto
- **Фирменные ярлыки**: капс, увеличенный `letter-spacing`

### Ключевые блоки
- Полосатая верхняя плашка
- Заголовок «КАК ПРАВИЛЬНО ИГРАТЬ»
- 6 карточек рекомендаций с нумерацией
- Модальный тест с расчетом уровня риска
