# Шансы на титул | Спецпроект Спортс.ру

Современный интерактивный каталог спортивных чемпионов с 2000 года. Красивый дизайн в стиле Sports.ru с плавными анимациями, умными фильтрами и адаптивной версткой.

![Champions Project](https://dumpster.cdn.sports.ru/d/37/92a0c4614cb5541f90d25dcdd80d3.webp)

## 🎯 Описание

Спецпроект представляет собой современную галерею спортивных чемпионов с множественной фильтрацией по различным параметрам. Дизайн выполнен в фирменном стиле Sports.ru с использованием брендовых цветов и типографики.

## ✨ Основные возможности

### 🎨 Современный дизайн
- **Градиентный hero-блок** с анимированной статистикой
- **Плавные переходы** и анимации появления элементов
- **Карточки с hover-эффектами**: увеличение изображения, тени, трансформации
- **Адаптивная типографика** от мобильных до десктопных экранов
- **Индикаторы вероятности** в виде прогресс-баров

### 🔍 Умные фильтры
- **Поиск по имени** чемпиона с мгновенными результатами
- **Фильтрация по виду спорта**: Формула-1, Дартс, Футбол, Хоккей и другие
- **Фильтрация по турнирам**: Олимпиады, Чемпионаты мира, Кубки
- **Фильтрация по годам**: от 2000 до 2025
- **Sticky-позиционирование** фильтров при прокрутке
- **Кнопка сброса** всех фильтров
- **Мобильная версия** фильтров с выдвигающейся панелью

### 📊 Режимы отображения
- **Коэффициенты** (например: 2.75)
- **Вероятности** (например: 36.36%)
- Переключение одной кнопкой с сохранением фильтров

### 🎴 Красивые карточки
- **Качественные изображения** чемпионов
- **Бейджи с коэффициентами** с анимацией пульсации
- **Метаданные**: вид спорта, турнир, дата
- **Разворачивающееся описание** с плавной анимацией
- **Прогресс-бар вероятности** внизу каждой карточки

### 📱 Полная адаптивность
- **Desktop** (1024px+): сетка карточек, все фильтры видны
- **Tablet** (768px-1023px): адаптированные карточки
- **Mobile** (до 767px): вертикальные карточки, выдвижные фильтры
- **Small Mobile** (до 480px): оптимизированные размеры

## 🎬 Анимации

- **Fade-in** эффекты при загрузке страницы
- **Slide-up** анимация для hero-блока
- **Pulse** анимация для бейджей с коэффициентами
- **Transform** эффекты при hover на карточках
- **Smooth transitions** для всех интерактивных элементов
- **Animated numbers** в статистике hero-блока

## 📊 База данных

### Текущая статистика:
- **52 чемпиона** из разных видов спорта
- **2 вида спорта**: Формула-1, Дартс
- **2 турнира**: Формула-1 (2000-2025), ЧМ по дартсу PDC (2000-2025)
- **Период**: 2000-2025 годы

### Формат данных:
```javascript
{
    id: 'unique-id',
    name: "Имя чемпиона",
    year: 2024,
    date: '01.01.2024',
    sport: 'motorsport',        // motorsport, darts, football, etc.
    tournament: 'formula1',      // formula1, darts_wc, olympics, etc.
    description: "Полное описание достижения...",
    odds: 2.75,                  // Коэффициент
    probability: 36.36,          // Вероятность (%)
    image: "https://..."         // URL изображения
}
```

## 🔧 Разработка

### Добавление нового вида спорта:
1. Добавить в `data/all-champions.js` записи с новым `sport`
2. Обновить словарь `SPORT_NAMES` в `index.html`
3. Фильтр автоматически обновится

### Добавление нового турнира:
1. Добавить записи с новым `tournament`
2. Обновить словарь `TOURNAMENT_NAMES`
3. Готово!

### Изменение цветовой схемы:
```css
/* В :root изменить переменные */
--sports-primary-color: #ваш-цвет;
--sports-primary-color-hover: #ваш-hover-цвет;
```
