# Kalshi Odds Trajectory

## Описание
Анимированный виджет с графиком коэффициентов по датам: логотипы команд перемещаются слева направо и по вертикали отражают значение коэффициента. Внизу — ось дат, слева — ось коэффициентов.

## Возможности
- Анимация движения логотипов по точкам дат.
- Линии тренда для каждой команды.
- Поддержка логарифмического масштаба по оси коэффициентов (по умолчанию включен).
- Курсор текущей позиции на линии (в реальном времени).
- Подсказка при наведении/касании по текущему положению.
- Перемотка: потяните по графику, чтобы переместиться к нужной дате.
