# Калькулятор вилок (арбитражных ставок)

Виджет для расчета арбитражных ставок на спортивные события в стиле Sports.ru

## Описание

Калькулятор вилок позволяет:
- Определить наличие арбитражной ситуации (вилки) между коэффициентами разных букмекеров
- Рассчитать процент прибыли от арбитража
- Распределить общую сумму ставки по исходам для получения гарантированной прибыли
- Работать с событиями на 2 или 3 исхода

## Функциональность

### Основные возможности
1. **Расчет вилки**: Определение наличия арбитража по формуле `1/К1 + 1/К2 [+ 1/К3] < 1`
2. **Расчет прибыли**: Показывает процент гарантированной прибыли
3. **Распределение ставок**: Автоматический расчет сумм ставок на каждый исход
4. **Визуальная индикация**: 
   - Зеленый цвет для прибыльных вилок
   - Красный цвет при отсутствии арбитража

### Интерфейс
- **Переключатель типа события**: 2 или 3 исхода
- **Поля коэффициентов**: Ввод коэффициентов от разных букмекеров
- **Поле общей суммы**: Сумма для распределения между ставками
- **Блок результатов**: Показывает процент прибыли и распределение ставок

## Как работает арбитраж

### Принцип работы
Арбитражная ставка возможна, когда сумма обратных коэффициентов меньше единицы:
```
1/К1 + 1/К2 < 1 (для двух исходов)
1/К1 + 1/К2 + 1/К3 < 1 (для трех исходов)
```

### Расчет ставок
Для получения одинаковой прибыли на каждый исход:
```
Ставка на исход = (Общая сумма / Коэффициент) / Сумма обратных коэффициентов
```

## Пример использования

1. **Футбольный матч**:
   - П1: 2.10 (букмекер А)
   - Х: 3.40 (букмекер Б)  
   - П2: 4.20 (букмекер В)

2. **Проверка вилки**:
   ```
   1/2.10 + 1/3.40 + 1/4.20 = 0.476 + 0.294 + 0.238 = 1.008
   ```
   Вилки нет (сумма > 1)

3. **Выгодные коэффициенты**:
   - П1: 2.05
   - Х: 3.50
   - П2: 4.50
   
   ```
   1/2.05 + 1/3.50 + 1/4.50 = 0.488 + 0.286 + 0.222 = 0.996
   ```
   Есть вилка! Прибыль: 0.4%
