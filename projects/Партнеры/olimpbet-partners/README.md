# Партнеры Olimpbet

## Описание
Виджет отображения партнеров букмекерской компании Olimpbet. Показывает логотипы и названия спортивных команд, лиг и организаций, являющихся партнерами компании.

## Особенности
- Адаптивная сетка партнеров
- Поддержка темной и светлой темы
- Оптимизированная загрузка изображений
- Современный дизайн в стиле Sports.ru
- Автоматическое изменение размера iframe
