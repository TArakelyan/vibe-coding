# Амбассадоры российских букмекеров (bookmaker-ambassadors-ratings)

## Описание

Интерактивная сводная страница по амбассадорам российских букмекеров: минималистичный интерфейс в духе рейтинговых списков, базовая ширина контейнера **660px** как в правилах встраивания Sports.ru, без фирменного визуала Sports (нейтральная типографика и серые акценты). Сверху сразу поиск и фильтры, без отдельного текстового заголовка страницы.

## Возможности

- Поиск по имени и деятельности.
- Выпадающие фильтры (нативные `<select>`): букмекер, деятельность.
- Сброс фильтров одной кнопкой.
- Сортировка по клику на заголовки: букмекер (алфавит), имя, деятельность.
- Строка таблицы: в первой колонке логотип букмекера в круге (картинка на всю площадь круга), рядом название бренда; имя амбассадора: **фамилия жирным**, имя обычным (одно слово — целиком жирным).
- Файл `resize.js` для корректной высоты iframe на площадке.

## Деплой (dumpster, один HTML-файл)

На CDN часто публикуется только **`index.html`** без соседних `styles.css` и `js` — тогда относительные пути не работают.

- В репозитории **`index.html` уже собранный**: внутри него встроены стили и все скрипты (подходит для деплоя одним файлом).
- Разметка с внешними `./styles.css` и `./data/...` лежит в **`index.source.html`** — её правьте при изменении структуры страницы.
- После правок в `index.source.html`, `styles.css`, `main.js` или `data/*` выполните в папке проекта: **`node build-standalone.mjs`** — скрипт заново запишет **`index.html`** (и копию **`standalone.html`**).
- Локально для отладки с отдельными файлами откройте **`index.source.html`** (нужен локальный сервер или открытие из той же папки, чтобы подтянулись относительные ресурсы).

## Данные

- Список амбассадоров: `data/ambassadors.js` (в т.ч. поля `primaryNetwork` и `followers` для возможного возврата колонки соцсетей позже).
- Логотипы букмекеров: `data/bookmakers.js` (копия URL из `projects/social-betting/data/bookmakers-social.js`).
- Файл `data/social-networks.js` в репозитории сохранён для будущего использования, в `index.html` не подключается.
