# Амбассадоры российских букмекеров (bookmaker-ambassadors-ratings)

## Описание

Интерактивная сводная страница по амбассадорам российских букмекеров: минималистичный интерфейс в духе рейтинговых списков (логика фильтров и чистая сетка строк), базовая ширина контейнера **660px** как в правилах встраивания Sports.ru, без фирменного визуала Sports (нейтральная типографика и серые акценты).

## Возможности

- Поиск по имени и деятельности.
- Выпадающие фильтры: букмекер, деятельность, основная соцсеть.
- Сброс фильтров одной кнопкой.
- Сортировка по клику на заголовки: букмекер (алфавит), имя, деятельность, соцсеть, число подписчиков.
- Строка таблицы: логотип букмекера (URL из проекта `social-betting`), фото и имя на двух строках, деятельность, иконка соцсети и «пилюля» с числом подписчиков.
- Логотип Sports.ru в подвале ведёт на sports.ru с UTM `special-bookmaker-ambassadors-ratings`.
- Файл `resize.js` для корректной высоты iframe на площадке.

## Данные

- Список амбассадоров: `data/ambassadors.js` — поля `followers` (число подписчиков в основной соцсети) и при необходимости `primaryNetwork` нужно **актуализировать вручную**; сейчас у большинства записей `followers: null` (в таблице отображается «—»), а `primaryNetwork` задан эвристически для корректной работы фильтра по соцсети.
- Логотипы букмекеров: `data/bookmakers.js` (копия URL из `projects/social-betting/data/bookmakers-social.js`).
- Иконки сетей: `data/social-networks.js` (Telegram, VK, YouTube, MAX — dumpster CDN; Instagram, Twitch, TikTok — встроенные SVG).
