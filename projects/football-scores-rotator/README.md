# Ротатор Футбольных Счетов

## Описание

Интерактивный виджет-ротатор с прогнозами на матч «Бавария» - «Байер» (Леверкузен) с адаптивным дизайном. Пользователь может запустить ротацию изображений с различными прогнозами, которая случайным образом остановится на одном из них. После остановки появляется кнопка для перехода на официальный сайт Winline. Виджет автоматически подстраивается под размер контейнера.

## Особенности

- **Адаптивный дизайн**: Виджет автоматически подстраивается под размер контейнера без фиксированных размеров
- **Современная структура**: Кнопки и плашка топ-матча расположены под изображением внутри виджета
- **Интерактивная ротация**: Быстрая смена изображений с плавным замедлением
- **Случайная остановка**: Алгоритм рандомной остановки на одном из 5 изображений
- **Анимированные кнопки**: Красивые кнопки с эффектами наведения и нажатия
- **Мобильная адаптивность**: Корректное отображение на всех устройствах
- **Интеграция с Winline**: Прямая ссылка на официальный сайт букмекера
- **Текстовые описания**: Каждое изображение имеет уникальный заголовок и описание матча

## Функциональность

### Основные возможности:
- Ротация 5 изображений с прогнозами на матч «Бавария» - «Байер»
- Уникальные прогнозы: победа «Баварии», голы Кейна, сенсация «Байера», промах Нойера, скучная ничья
- Кнопка "КРУТИТЬ" для запуска ротации
- Плавное замедление и остановка на случайном изображении
- Появление кнопки "ПЕРЕЙТИ НА WINLINE" после остановки
- Возможность повторного запуска ротации
- Статусные сообщения для пользователя

### Технические особенности:
- Предзагрузка всех изображений для плавной работы
- Класс-контроллер для управления состоянием виджета
- Анимации CSS с аппаратным ускорением
- Обработка ошибок загрузки изображений
- Автоматическое изменение размера iframe

## Файлы проекта

- `index.html` - Основная HTML разметка
- `styles.css` - Стили и анимации
- `script.js` - Логика ротации и интерактивности
- `resize.js` - Скрипт для корректного отображения в iframe
- `README.md` - Документация проекта

## Использование

1. Откройте `index.html` в браузере
2. Нажмите кнопку "КРУТИТЬ" для запуска ротации
3. Дождитесь остановки на случайном изображении
4. Нажмите "ПЕРЕЙТИ НА WINLINE" для перехода на сайт букмекера
5. При желании повторите процесс

## Интеграция

Для встраивания в пост используйте следующий код:

```html
<iframe class="auto-height" height="400" id="special-project-football-scores-rotator" src="ССЫЛКА_НА_ВИДЖЕТ" style="border: 0px;" width="100%"></iframe>
<iframe class="iframe-video" height="0" src="/picker/resize/" style="height: 0px; display:none;" width="730"></iframe>
```

Виджет автоматически подстроится под размер контейнера благодаря адаптивному дизайну.
