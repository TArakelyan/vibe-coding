# Подписчики букмекеров в соцсетях

Интерактивный виджет Sports: сравнение аудиторий российских букмекеров в Telegram, VK, Max и YouTube. Данные в тысячах подписчиков; при отсутствии данных показывается прочерк.

## Особенности

- Четыре переключателя соцсетей с фирменными иконками (как по размеру у прежних табов «по годам»).
- Одновременно активна только одна сеть; список букмекеров обновляется под выбранную площадку.
- Сортировка по убыванию подписчиков; строки без данных — в конце списка.
- Оформление в стиле Sports: шрифты Sports и Roboto, брендовые цвета, адаптация к светлой и тёмной теме.
- Без модулей: данные в `data/bookmakers-social.js`, подключение через относительные пути.
- Автоподстройка высоты iframe (`resize.js`).
