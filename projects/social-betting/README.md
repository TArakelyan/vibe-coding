# Подписчики букмекеров в соцсетях

Виджет `social-betting` обновлен: вместо годов и карточек финансовой отчетности теперь отображается аудитория букмекеров по соцсетям.

## Описание

Сверху расположены четыре категории с логотипами: `Telegram`, `VK`, `Max`, `Youtube`. Можно выбрать только одну категорию.  
Ниже выводится список букмекеров и количество подписчиков (`fol`) для выбранной сети.
Для категорий `Telegram` и `Max` дополнительно выводится колонка `ER`.

## Особенности

- Визуальный стиль проекта сохранен (та же стилистика табов и блоков).
- Переключение только между одной активной категорией.
- Сортировка по убыванию подписчиков в выбранной соцсети.
- Отсутствующие данные отображаются как прочерк `—`.
- Для `Telegram` и `Max` поддерживается дополнительный показатель вовлечённости `ER`.
- У каждого букмекера в списке показывается логотип в круге с лёгким «зумом» и обрезкой, чтобы убрать белые поля вокруг картинки; иконка VK в табах — актуальный файл с CDN.
- Данные вынесены в `data/bookmakers-social.js`.
