# ОлимпБет Widget

## Описание
Рекламный виджет ОлимпБет с современным дизайном и анимациями

## Особенности
- **Desktop**: 672px × 72px (при ширине экрана >500px)
- **Mobile**: 288px × 108px (при ширине экрана ≤500px)
- Анимированная кнопка с pulse эффектом в фирменном красном цвете (#F42525)
- Hover эффекты
- Ripple анимация при клике на кнопку
- Ротация текстов для разнообразия контента
- Адаптивный дизайн для мобильных устройств

## Компоненты
- **Логотип ОлимпБет**: размещен слева с оптимизированным рендерингом
- **Плашка "Реклама 18+"**: серый цвет с тремя точками для показа информации о рекламодателе
- **Текст**: ротация из 4 вариантов промо-текстов (шрифт Inter)
- **Кнопка**: красная (#F42525) с белой стрелкой и анимацией pulse

## Файлы
- `index.html` - HTML структура виджета с подключением внешних ресурсов и встроенным resize скриптом
- Внешние ресурсы:
  - CSS: https://dumpster.cdn.sports.ru/7/8e/d82c4fb64b5c00ac3733fe61fa605.css
  - JS (основной): https://dumpster.cdn.sports.ru/5/b6/6d6d005e216ef4037a528d8fc6421.js
  - Логотип: https://dumpster.cdn.sports.ru/3/f5/283734791d928a5d5e74dc539cbe9.png
- Встроенный resize скрипт для правильного определения высоты виджета

## Функциональность
- Клик по виджету открывает https://spnsrd.ru/pst2olimp
- Показ информации о рекламодателе при наведении на три точки
- Автоматическая ротация промо-текстов при загрузке
- Адаптивное изменение размера iframe через postMessage
