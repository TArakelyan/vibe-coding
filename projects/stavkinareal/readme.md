# StavkiNaReal — Анализ ставок на Реал Мадрид

## Описание
Интерактивный виджет для анализа результатов ставок на матчи Реала. Можно выбрать стратегию (победа/ничья/поражение, не проиграет/не выиграет), задать фиксированную сумму ставки и увидеть итоговую прибыль/убыток по последним 10 матчам.

## Особенности
- Современный UI в стиле Sports.ru (макс. ширина 660px)
- Переключение стратегий с мгновенным пересчетом результатов
- Гибкий выбор суммы или ввод собственной
- Подпись таблицы: «Результаты последних игр Реала»
- Плашка-предупреждение о ответственном гемблинге
- Кнопка «Повторить ставку с WINLINE» со ссылкой `https://spnsrd.ru/tag/teams/freebet` и цветом #FF6913
- Автоматический ресайз и авто-детектор темы
- **10 последних матчей** вместо 6 для более детального анализа
