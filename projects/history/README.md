# Спортс — архив спортивной истории

## Описание

«Спортс» — одностраничное приложение для изучения спортивной истории: разделы «Википедия» (статьи), «Библиотека» (книги), «Кинотеатр» (фильмы), переключаемые без перезагрузки. Визуально — спортивный портал в духе [FOX Sports](https://www.foxsports.com/), с **тёмно-зелёной** палитрой акцентов (шапка чёрная, ленты и кнопки — зелёные оттенки). Контент задаётся в `data/content.js`.

## Особенности

- **Бренд «Спортс»** — единый логотип-слово в шапке, без старых подписей Sports History.
- **Оформление FOX Sports + зелёная тема** — шрифты Oswald + Source Sans 3, ленты разделов, прямоугольные кнопки; акцентный цвет — зелёный вместо синего.
- **Обзор книги** — карточка в библиотеке открывает страницу с хлебными крошками, тегами, обложкой и метаданными, блоками «Предпросмотр», «Рецензия» и формой «Оценки». Для книг без полей в данных текст предпросмотра и рецензии собирается автоматически из описания; расширенный пример — запись `open-agassi` в `content.js`.
- **Страница фильма / сериала** — из сетки «Кинотеатр»: формат, режиссёр, жанр, год, длительность, страна, возрастной рейтинг, оценки Кинопоиска и IMDb, редакционная оценка Спортс, сценарий, актёры; блоки «Предпросмотр», **«Трейлер»** (контейнер 16:9: встроенное видео при полях `trailerYoutubeId` или `trailerEmbedUrl` в данных, иначе заглушка), «Рецензия» и «Оценки» с формой.
- **Глобальный поиск в шапке** — от двух символов показывает совпадения по статьям, книгам и фильмам; клик открывает нужный материал. Горячая клавиша **/** фокусирует поле поиска.
- **Фильтры «Другие»** — в разделах «Википедия», «Библиотека» и «Кинотеатр» после ряда чипов фильтра добавлена вкладка **«Другие»**: раскрывающийся список со ссылками на разводящие страницы (`#section-landing`). Тексты и привязка к фильтрам задаются в `data/content.js` в объекте `othersDropdownLandings`.
- **Разделы без дублирующей ленты** — название раздела только в меню и в заголовке страницы; под заголовком — блок `.section-page-lead` с коротким описанием назначения раздела.
- **Ширина контента** — без `max-width` у контейнера; на десктопе (от 1024px) у страницы добавлены боковые поля `clamp(50px, 7vw, 100px)`, чтобы контент не прилипал к краям окна.
- **Навигация без «гамбургера»** — пункты меню всегда в шапке; на узком экране доступна горизонтальная прокрутка.
- **Адаптивные сетки** карточек статей, книг и фильмов, фильтры по категориям в библиотеке.
