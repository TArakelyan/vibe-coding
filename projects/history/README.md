# Спортс — архив спортивной истории

## Описание

«Спортс» — одностраничное приложение для изучения спортивной истории: разделы «Википедия» (статьи), «Библиотека» (книги), «Кинотеатр» (фильмы), переключаемые без перезагрузки. Визуальный язык ориентирован на порталы в духе [FOX Sports](https://www.foxsports.com/): тёмная шапка, синие акценты, навигация и карточки в стиле спортивных медиа. Контент по-прежнему задаётся в `data/content.js`.

## Особенности

- **Бренд «Спортс»** — единый логотип-слово в шапке, без старых подписей Sports History.
- **Оформление FOX Sports** — шрифты Oswald + Source Sans 3, палитра чёрный / тёмно-синий / синий FOX, ленты разделов и прямоугольные кнопки.
- **Полная ширина макета** — контейнеры без ограничения `max-width`, только отступы по краям.
- **Навигация без «гамбургера»** — пункты меню всегда в шапке; на узком экране доступна горизонтальная прокрутка.
- **Адаптивные сетки** карточек статей, книг и фильмов, фильтры по категориям в библиотеке.
