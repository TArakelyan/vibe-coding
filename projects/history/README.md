# Спортс — архив спортивной истории

## Описание

«Спортс» — одностраничное приложение для изучения спортивной истории: разделы «Википедия» (статьи), «Библиотека» (книги), «Кинотеатр» (фильмы), **«Сегодня»** (дайджест событий «в этот день» из истории котировок), переключаемые без перезагрузки. Визуально — спортивный портал в духе [FOX Sports](https://www.foxsports.com/), с **тёмно-зелёной** палитрой акцентов (шапка чёрная, ленты и кнопки — зелёные оттенки). Основной контент — в `data/content.js`, события раздела «Сегодня» — в `data/today-events.js`.

## Особенности

- **Бренд «Спортс»** — единый логотип-слово в шапке, без старых подписей Sports History.
- **Оформление FOX Sports + зелёная тема** — шрифты Oswald + Source Sans 3, ленты разделов, прямоугольные кнопки; акцентный цвет — зелёный вместо синего.
- **Обзор книги** — карточка в библиотеке открывает страницу с хлебными крошками, тегами, обложкой и метаданными, блоками «Предпросмотр», «Рецензия» и формой «Оценки». Для книг без полей в данных текст предпросмотра и рецензии собирается автоматически из описания; расширенный пример — запись `open-agassi` в `content.js`. **Читай-город** и **Литрес** оформлены как яркие капсовые «пилюли» с иконкой книг (оранжевая и синяя ссылки), в сетке — в светлой рамке; URL по умолчанию — поиск по русскому названию, либо поля `chitaiGorodUrl` и `litresUrl` в данных.
- **Сетки карточек (главные страницы разделов)** — **Википедия:** только обложка 16:9, заголовок и число лайков; фильтры — чипы «Все / Чемпионы / Финансы», выпадающие «Виды спорта» и «По категориям», пункт «Другое» (лендинги). На карточках для отладки выставляются `data-wiki-*` из полей `wikiPillar`, `wikiSport`, `wikiTaxonomy` в данных статьи. **Библиотека и Кинотеатр:** на узком экране — 2 колонки, от `md` — 3, от `lg` — **4 колонки** (афиша и обложка ниже по высоте); под обложкой — русское название, год и страницы / оценка и режиссёр, пилюли магазинов или «Смотреть».
- **Сетка «Кинотеатр»** — карточки с афишами в соотношении сторон **2:3**, как обложки книг в «Библиотеке» (общий блок `.book-cover-wrapper`).
- **Страница фильма / сериала** — из сетки «Кинотеатр»: формат, режиссёр, жанр, год, длительность, страна, возрастной рейтинг, оценки Кинопоиска и IMDb, редакционная оценка Спортс, сценарий, актёры; блоки «Предпросмотр», **«Трейлер»** (контейнер 16:9: встроенное видео при полях `trailerYoutubeId` или `trailerEmbedUrl` в данных, иначе заглушка), «Рецензия» и «Оценки» с формой.
- **Глобальный поиск в шапке** — компактное поле справа после пунктов меню (иконка лупы и ввод, без подписи «Поиск»); от двух символов показывает совпадения по статьям, книгам и фильмам. Горячая клавиша **/** фокусирует поле.
- **Раздел «Сегодня»** — заголовок, лид, чипы (**Все**, **Матчи**, **Явления**, **Даты**, **Трагедии**, **Правила**, **Награды**), выпадающий список **«Виды спорта»** и **«Другое»** (лендинги); полоска из трёх дней с оранжевой отметкой активного дня и стрелками ‹ ›; карточки событий — фото и текстовый блок: бейдж **года** оранжевым блоком центрируется по **горизонтали всей карточки** у верхнего края (наполовину над блоком события); текст с акцентными ссылками; внешняя ссылка с подписью **«Читать об этом»**. В данных событий задаются `todayKind` и `todaySport`; демо для **3 мая** — в `data/today-events.js` (`todayEventsByMonthDay`, `todayDemoAnchorMonthDay`).
- **Фильтры «Другое»** — в разделах «Википедия», «Библиотека», «Кинотеатр» и **«Сегодня»** после ряда чипов и выпадашек — пункт **«Другое»**: раскрывающийся список со ссылками на разводящие страницы (`#section-landing`). Тексты и привязка к фильтрам задаются в `data/content.js` в объекте `othersDropdownLandings` (`filterPillar`, `libraryCategory`, `cinemaCategory`, `todayKind` / `todaySport`).
- **Разделы без дублирующей ленты** — название раздела только в меню и в заголовке страницы; под заголовком — блок `.section-page-lead` с коротким описанием назначения раздела.
- **Ширина контента** — без `max-width` у контейнера; на десктопе (от 1024px) у страницы добавлены боковые поля `clamp(50px, 7vw, 100px)`, чтобы контент не прилипал к краям окна.
- **Навигация без «гамбургера»** — пункты меню всегда в шапке; на узком экране доступна горизонтальная прокрутка.
- **Мобильная шапка и фильтры** — до ~900px: сетка «логотип + строка поиска», меню ниже на всю ширину (прокрутка без обрезания подписей); чипы фильтров в разделах — выровненная сетка из двух колонок, кнопка «Другие» на всю ширину с панелью без вылезания за экран.
- **Адаптивные сетки** карточек статей, книг и фильмов; в библиотеке — фильтры по полке (жанр/фокус) и виду спорта.
