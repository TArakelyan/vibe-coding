# Виджет статистики ставок (минимализм)

## Описание
Компактный адаптивный UI-виджет для отображения статистики последних 5 прогнозов.  
Компонент выполнен в минималистичном стиле, использует светлый фон, аккуратные скругления и четкую визуальную иерархию текста.

## Возможности
- Числа задаются в `data/widget-data.js` и подключаются в `index.html` (без дублирования инлайн-данных).
- Десктопный формат: компактная горизонтальная строка с заголовком, иконками и средним кэфом.
- Мобильный формат: вертикальная компоновка с аккуратным распределением блоков по ширине.
- Цветовые индикаторы результатов (+ / - / =) в круглых иконках по 20px.
- Выделение ключевых значений жирным начертанием для быстрого чтения.
- Резиновое поведение при ширине экрана меньше 320px (перенос блока с кэфом).
- Поддержка авто-ресайза для iframe через `resize.js`.
- Готовность к деплою одним HTML: стили и скрипты подключаются по CDN, данные могут задаваться inline в `index.html`.
