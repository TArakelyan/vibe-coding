# 30 лучших дотеров всех времен

## Описание

Интерактивный рейтинг 30 лучших игроков в Dota 2 всех времен по версии экспертов. Проект представляет собой полную копию страницы cyber.sports.ru с информацией о легендарных киберспортсменах, их достижениях и вкладе в историю Dota 2.

## Особенности

### Основной функционал
- **Интерактивные карточки игроков** - каждая карточка содержит рейтинг, никнейм, реальное имя и фото игрока
- **Модальные окна** - подробная информация об игроке при клике на карточку
- **Анимации** - плавные появления элементов при скролле страницы
- **Hover-эффекты** - карточки поднимаются и подсвечиваются при наведении
- **Parallax эффект** - динамичный hero-раздел с эффектом параллакса

### Раздел жюри
- **Система табов** - переключение между категориями экспертов (Эксперты, Игроки, Комментаторы, Редакторы)
- **Аватары жюри** - отображение инициалов членов жюри
- **Описание методологии** - подробное объяснение системы подсчета рейтинга

### Технические особенности
- **Адаптивный дизайн** - корректное отображение на всех устройствах
- **Максимальная ширина 660px** - оптимизация для встраивания в посты
- **Resize.js** - автоматическая подстройка высоты iframe при встраивании
- **Модульная структура** - данные вынесены в отдельные JS файлы
- **Брендированный дизайн Sports.ru** - использование фирменных цветов и шрифтов

### Интерактивность
- **Клик по карточке** - открывает модальное окно с деталями
- **ESC** - закрывает модальное окно
- **Плавная прокрутка** - для якорных ссылок
- **Кнопка "Собери свой топ"** - призыв к действию для пользователей

## Структура проекта

```
dota-top-30/
├── index.html          # Основная HTML структура
├── styles.css          # Стили с использованием CSS переменных Sports.ru
├── script.js           # Основная логика и интерактивность
├── resize.js           # Скрипт для корректной работы в iframe
├── data/
│   ├── players.js      # Данные о 30 лучших игроках
│   └── jury.js         # Данные о членах жюри
└── README.md           # Документация проекта
```

## Использованные технологии

- **HTML5** - семантическая разметка
- **CSS3** - современные стили с переменными, Grid и Flexbox
- **Vanilla JavaScript** - без использования фреймворков
- **Современный CSS** - Grid, Flexbox, CSS переменные
- **Intersection Observer API** - оптимизированные анимации появления
- **ResizeObserver API** - отслеживание изменений размера для iframe

## Дизайн

Проект полностью повторяет оригинальный дизайн cyber.sports.ru:
- Акцентный цвет: `#00D9FF` (циановый)
- Шрифт: Inter (все элементы)
- Темная цветовая схема (#1a1a1a фон, #2a2a2a карточки)
- Широкоформатная раскладка без жестких ограничений ширины
- 3-колоночная сетка на desktop, адаптивная на мобильных
