📁 ОСНОВНЫЕ МОДУЛИ (Python)
═══════════════════════════════════════════════════════════════════════════

1. main.py (200 строк)
   └─→ Главный файл запуска
   └─→ APScheduler для расписания
   └─→ Координация всех модулей
   └─→ Логирование

2. parser.py (350 строк)
   └─→ Основной парсер новостей
   └─→ AsyncIO для параллельной обработки
   └─→ BeautifulSoup + feedparser
   └─→ Фильтрация по ключевым словам
   └─→ Дедупликация через Redis

3. source_discovery.py (300 строк)
   └─→ Динамический поиск источников
   └─→ Google/Yandex/Dzen поиск
   └─→ Фильтрация релевантных доменов
   └─→ Расчет рейтинга надежности

4. ml_classifier.py (250 строк)
   └─→ ML/NLP классификация
   └─→ spaCy для русского языка
   └─→ Извлечение сущностей (NER)
   └─→ Анализ тональности
   └─→ TF-IDF + Naive Bayes

5. database.py (300 строк)
   └─→ PostgreSQL операции
   └─→ CRUD для новостей/источников
   └─→ Статистика и аналитика
   └─→ Connection pooling

6. telegram_bot.py (150 строк)
   └─→ Telegram интеграция
   └─→ Форматирование сообщений
   └─→ Пакетная отправка
   └─→ Ежедневные сводки

7. dashboard.py (200 строк)
   └─→ Flask REST API
   └─→ Plotly графики
   └─→ Endpoints для дашборда
   └─→ JSON serialization

8. tasks.py (150 строк)
   └─→ Celery фоновые задачи
   └─→ Beat scheduler
   └─→ 5 типов задач
   └─→ Cron расписание

9. config.py (250 строк)
   └─→ Конфигурация проекта
   └─→ 200+ источников
   └─→ 100+ ключевых слов
   └─→ Все настройки

10. test_parser.py (200 строк)
    └─→ Тестирование модулей
    └─→ Проверка релевантности
    └─→ Тест ML классификатора
    └─→ Интеграционные тесты

11. setup_local.py (150 строк)
    └─→ Локальная установка
    └─→ Проверка зависимостей
    └─→ Инициализация окружения

   └─→ 7 сервисов:

   └─→ Volumes для persistence

   └─→ spaCy модель

3. nginx.conf (30 строк)
   └─→ Reverse proxy настройка
   └─→ Проксирование на dashboard:5000
   └─→ Заголовки для real IP
