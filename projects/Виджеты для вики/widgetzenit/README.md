# Виджет Зенит

## Описание
Интерактивный виджет для продвижения бонусных предложений Зенит. Виджет адаптивен и оптимизирован для всех устройств.

## Особенности
- Адаптивный дизайн для мобильных и десктопных устройств
- Анимированная кнопка действия с эффектом пульсации
- Hover-эффекты для улучшения пользовательского опыта
- Оптимизированная типографика с использованием шрифта Inter
- Автоматическое изменение размера iframe
- Плавные переходы и анимации

## Технические характеристики
- **Десктопная версия:** 672x72px
- **Мобильная версия:** адаптивная ширина (280-400px), высота 108px
- **Цвет бренда:** #011D3D (темно-синий Зенит)
- **Ссылка:** https://spnsrd.ru/wiki/posts/zenit
