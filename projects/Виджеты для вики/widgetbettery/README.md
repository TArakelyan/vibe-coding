# PARI Widget

## Описание
Рекламный виджет PARI с точной копией дизайна из референсов

## Особенности
- **Desktop**: 672px × 72px (при ширине экрана >500px)
- **Mobile**: 288px × 108px (при ширине экрана ≤500px)
- Анимированная кнопка с pulse эффектом
- Hover эффекты
- Ripple анимация при клике на кнопку
- Полностью кастомный дизайн без брендинга Sports.ru

## Компоненты
- **Логотип PARI**: обновленное изображение с оптимизированным рендерингом
- **Плашка "Реклама 18+"**: серый цвет в правом верхнем углу, шрифт Inter
- **Текст**: динамический текст с ротацией из 4 вариантов (шрифт Inter)
- **Кнопка**: черная с белой стрелкой и анимацией
