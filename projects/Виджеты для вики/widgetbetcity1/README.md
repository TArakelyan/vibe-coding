# BETCITY Widget

## Описание
Рекламный виджет BETCITY с обновленным дизайном и брендингом

## Особенности
- **Desktop**: 672px × 72px (при ширине экрана >500px)
- **Mobile**: 288px × 108px (при ширине экрана ≤500px)
- Анимированная кнопка с pulse эффектом в фирменном синем цвете
- Hover эффекты
- Ripple анимация при клике на кнопку
- Полностью кастомный дизайн без брендинга Sports.ru

## Компоненты
- **Логотип BETCITY**: новый логотип с оптимизированным рендерингом (betcitylogo.png)
- **Плашка "Реклама 18+"**: серый цвет в правом верхнем углу, шрифт Inter
- **Текст**: "➡️ Эксклюзивный промокод на три фрибета" (шрифт Inter)
- **Кнопка**: синяя (#0173B7) с белой стрелкой и анимацией

## Последние обновления
- **Изменен цвет кнопки**: новый фирменный синий цвет #0173B7
- **Заменен логотип**: обновлен на новый логотип BETCITY (betcitylogo.png)
- **Изменен цвет контура**: рамка виджета теперь синяя (#0173B7)
- **Обновлены ссылки**: переход на https://spnsrd.ru/pst1betcity
- **Обновлена цветовая схема анимаций**: pulse эффект теперь использует синий цвет
- **Поднят логотип на десктопе**: логотип BETCITY поднят на 8px вверх для лучшего позиционирования (только десктопная версия)
