# BETCITY Widget - Выход в 1/4 финала

Виджет для отображения коэффициентов на выход в 1/4 финала.

## Описание

Виджет отображает:
- Заголовок "Выход в 1/4 финала"
- Две команды с логотипами и коэффициентами:
  - Реал (3.20)
  - Ман Сити (1.36)
- Кнопку перехода на сайт партнера

## Особенности

- Все ссылки ведут на https://s.betcity.ru/a9x
- Адаптивный дизайн для десктопа и мобильных устройств
- Пустое пространство вокруг виджета (padding: 20px на десктопе, 10px на мобильных)
- Информация о рекламодателе при наведении на три точки

## Размеры виджета

### Десктопная версия
- Ширина виджета: 660px (max-width)
- Общая ширина с отступами: 700px (660px + 20px * 2)
- Высота виджета: ~230px (зависит от контента)
- Общая высота с отступами: ~270px

### Мобильная версия (до 500px)
- Ширина: 100% экрана
- Общая ширина с отступами: calc(100% - 20px) (padding: 10px с каждой стороны)
- Высота виджета: ~210px (зависит от контента)
- Общая высота с отступами: ~230px

## Файлы

- `index.html` - основная структура виджета
- `styles.css` - стили виджета
- `script.js` - интерактивность виджета
- `resize.js` - скрипт для изменения размера iframe
