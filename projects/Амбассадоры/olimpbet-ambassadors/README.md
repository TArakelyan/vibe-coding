# Амбассадоры Olimpbet

## Описание
Интерактивный виджет с амбассадорами букмекерской компании Olimpbet. Отображает сетку карточек с фотографиями и информацией о каждом амбассадоре.

## Особенности
- Адаптивная сетка карточек амбассадоров
- Поддержка темной и светлой темы
- Автоматическое изменение размера для iframe
- Оптимизированная загрузка изображений
- Современный дизайн в стиле Sports.ru

## Структура проекта
```
olimpbet-ambassadors/
├── index.html          # Основная страница
├── styles.css          # CSS стили
├── theme-detector.js   # Детектор темной темы
├── resize.js          # Скрипт автоподгонки размера
└── README.md          # Документация
```

## Технические особенности
- Чистый HTML/CSS/JavaScript без фреймворков
- Автоматическое определение темной темы системы
- Адаптивный дизайн для всех устройств
- Оптимизированная загрузка изображений
- Интеграция с системой автоподгонки размера iframe
