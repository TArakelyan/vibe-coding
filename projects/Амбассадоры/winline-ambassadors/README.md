# Амбассадоры WINLINE

## Описание
Интерактивный виджет с амбассадорами букмекерской компании WINLINE. Отображает сетку карточек с фотографиями и информацией о каждом амбассадоре.

## Особенности
- Адаптивная сетка карточек амбассадоров
- Поддержка темной и светлой темы
- Автоматическое изменение размера для iframe
- Оптимизированная загрузка изображений
- Современный дизайн в стиле Sports.ru
