# Амбассадоры Leon

## Описание
Виджет с карточками амбассадоров букмекерской компании Leon в стиле Sports.ru

## Особенности
- Адаптивная сетка карточек амбассадоров
- Поддержка тёмной и светлой темы с автоматическим определением
- Современный дизайн с использованием брендовых цветов Sports
- Hover-эффекты на карточках
- Оптимизированная загрузка изображений
- Данные амбассадоров вынесены в `data/ambassadors.js`

## Файлы
- `index.html` — разметка виджета
- `data/ambassadors.js` — данные амбассадоров
- `main.js` — отрисовка карточек
- `styles.css` — локальные стили (при необходимости)
- `theme-detector.js` — автоматическое определение темы Sports.ru
- `resize.js` — автоматическое изменение размера iframe
