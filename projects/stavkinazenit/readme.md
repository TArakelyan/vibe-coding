# StavkiNaZenit — Анализ ставок на Зенит

## Описание
Интерактивный виджет для анализа результатов ставок на матчи Зенита. Выберите стратегию (победа/ничья/поражение, не проиграет/не выиграет), задайте сумму и оцените суммарный результат по последним 6 играм.

## Особенности
- Современный UI в стиле Sports.ru (макс. ширина 660px)
- Переключение стратегий и пересчет результатов на лету
- Подпись таблицы: «Результаты последних игр Зенита»
- Плашка-предупреждение о ответственном гемблинге
- Кнопка «Повторить ставку с WINLINE» со ссылкой `https://spnsrd.ru/tag/teams/freebet` и цветом #FF6913
- Автоматический ресайз и авто-детектор темы
