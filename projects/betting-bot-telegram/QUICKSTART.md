## 📋 Что было сделано

### 1. Полнофункциональный бот на Python
- ✅ Асинхронная архитектура (aiogram 3.x)
- ✅ PostgreSQL база данных
- ✅ Интеграция с Google Sheets API
- ✅ Планировщик задач (APScheduler)
- ✅ Админ-панель с командами

### 2. Возможности бота

#### 🔄 Автоматический постинг
- **Бонус дня** - Суббота/Воскресенье (14:00, 18:00, 21:00)
- **Подборка офферов** - Среда (12:00)
- **Новые офферы** - Моментально при появлении
- **Лимит** - максимум 4 поста в день (настраивается)

#### 🎮 Ручное управление
```bash
/start                           # Главное меню
/publish_single bonus_123        # Опубликовать один бонус
/publish_bundle id1,id2,id3      # Подборка из 3 бонусов
/plan_week                       # План на неделю
/new_offers                      # Проверить новые офферы
/stats                           # Статистика
/sync_sheets                     # Синхронизация с Google Sheets
```

#### 🎨 Форматы постов
- **Одиночный пост**: картинка + текст + кнопка
- **Подборка**: общая картинка + 3 бонуса + 3 кнопки
- **Автоматические эмодзи**: случайный выбор из спортивных
- **Вступительные тексты**: рандомизация для разнообразия

## 🚀 Быстрый старт

### Шаг 1: Установка зависимостей

```bash
cd projects/betting-bot-telegram

# Создайте виртуальное окружение
python -m venv venv

# Активируйте (Windows)
venv\Scripts\activate

# Установите зависимости
pip install -r requirements.txt
```

### Шаг 2: Настройка конфигурации

1. Скопируйте `.env.example` в `.env`
2. Заполните необходимые параметры:

```env
# Telegram
BOT_TOKEN=ваш_токен_бота
CHANNEL_ID=@ваш_канал
ADMIN_IDS=ваш_telegram_id

# База данных PostgreSQL
DATABASE_URL=postgresql://user:password@localhost:5432/betting_bot

# Google Sheets
GOOGLE_SHEETS_ID=id_вашей_таблицы
GOOGLE_CREDENTIALS_FILE=credentials.json

# Настройки
MAX_POSTS_PER_DAY=4
ROTATION_DAYS=14
```

### Шаг 3: Настройка Google Sheets

1. Создайте Service Account в Google Cloud Console
2. Скачайте JSON ключ → переименуйте в `credentials.json`
3. Дайте доступ к таблице email из Service Account
4. Структура таблицы:

| bonus_id | title | end_date | post_url | image_url | category | bookmaker | amount | is_active |
|----------|-------|----------|----------|-----------|----------|-----------|--------|-----------|
| fonbet_1 | Фрибет 1000₽ | 2026-03-15 | https://... | https://... | promo | Fonbet | 1000₽ | TRUE |

**Подробнее см. [GOOGLE_SHEETS_FORMAT.md](GOOGLE_SHEETS_FORMAT.md)**

### Шаг 4: Инициализация базы данных

```bash
# Инициализация таблиц
python init_db.py

# Создать тестовые данные (опционально)
python create_test_data.py
```

### Шаг 5: Запуск бота

```bash
python bot.py
```

При успешном запуске вы увидите:

И получите уведомление в Telegram!

## 📊 Как получить необходимые данные

### Telegram Bot Token
1. Найдите [@BotFather](https://t.me/BotFather)
2. Отправьте `/newbot`
3. Следуйте инструкциям
4. Скопируйте токен

### Channel ID
- Для публичного канала: `@channel_username`
- Для приватного: `-100123456789` (можно узнать через @userinfobot)

### Admin ID
1. Найдите [@userinfobot](https://t.me/userinfobot)
2. Отправьте любое сообщение
3. Скопируйте ваш ID

### Google Sheets ID
Из URL таблицы:
```
https://docs.google.com/spreadsheets/d/[ВОТ_ЭТОТ_ID]/edit
```

## 🐛 Решение проблем

### Бот не отвечает
- Проверьте `BOT_TOKEN` в `.env`
- Убедитесь, что ваш ID в `ADMIN_IDS`
- Проверьте, что бот не запущен в другом месте

### Не синхронизируется Google Sheets
- Проверьте `credentials.json`
- Убедитесь, что Service Account имеет доступ к таблице
- Проверьте `GOOGLE_SHEETS_ID`

### Не публикуется в канал
- Добавьте бота как администратора канала
- Проверьте `CHANNEL_ID`
- Для приватных каналов используйте числовой ID

### Ошибка подключения к PostgreSQL
- Убедитесь, что PostgreSQL запущен
- Проверьте `DATABASE_URL` в `.env`
- База данных должна быть создана заранее
