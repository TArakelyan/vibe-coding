# Telegram-бот для постинга бонусов букмекеров

## 📋 Описание
Автоматизированный бот для публикации бонусов букмекерских контор из Sports.ru в Telegram-канал. Поддерживает автоматический постинг по расписанию и ручное управление через админ-панель.

## ✨ Возможности

### Типы постов
- **Бонус дня** - публикация под матчи еврокубков (суббота/воскресенье)
- **Эксклюзив Спортса** - посты с эксклюзивными промокодами
- **Новый оффер** - моментальная публикация новых бонусов
- **Подборка офферов** - еженедельная ротация из 3 бонусов

### Автоматизация
- ✅ Парсинг Google Sheets с данными о бонусах
- ✅ Автоматическая публикация по расписанию (cron)
- ✅ Ротация офферов (без повторов 14 дней)
- ✅ Умное планирование (max 4 поста/день)
- ✅ Интеграция с Yandex Disk для креативов

### Админ-панель
- `/start` - главное меню
- `/publish_single [ID]` - публикация одного бонуса
- `/publish_bundle [ID1,ID2,ID3]` - подборка из 3х бонусов
- `/plan_week` - план публикаций на неделю
- `/new_offers` - проверка новых офферов
- `/schedule` - текущее расписание

## 🛠 Технологии
- **Backend**: Python 3.11+
- **Bot Framework**: aiogram 3.x
- **База данных**: PostgreSQL
- **Планировщик**: APScheduler
- **API интеграции**: Google Sheets API, Yandex Disk API

## 📦 Установка

1. Клонируйте репозиторий
2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Создайте файл `.env` на основе `.env.example`
4. Настройте базу данных:
```bash
python init_db.py
```

5. Запустите бота:
```bash
python bot.py
```

## 🔧 Конфигурация

Все настройки в файле `.env`:
- `BOT_TOKEN` - токен Telegram-бота
- `CHANNEL_ID` - ID канала для публикаций
- `ADMIN_IDS` - ID администраторов (через запятую)
- `DATABASE_URL` - подключение к PostgreSQL
- `GOOGLE_SHEETS_ID` - ID Google таблицы
- `YANDEX_DISK_TOKEN` - токен Yandex Disk

## 📊 База данных

### Таблица `bonuses`
- `bonus_id` - уникальный ID бонуса
- `title` - название акции
- `end_date` - дата окончания
- `post_url` - ссылка на оффер
- `image_url` - ссылка на картинку (Dumpster)
- `category` - категория (eurocups, promo, new, bundle)
- `last_posted` - дата последней публикации

## 📅 Расписание публикаций

| Тип | Дни | Время | Приоритет |
|-----|-----|-------|-----------|
| Новый оффер | Любой | Моментально | 🔴 Максимальный |
| Бонус дня | Сб/Вс | 14:00, 18:00, 21:00 | 🟡 Высокий |
| Эксклюзив | Любой | По наличию | 🟢 Средний |
| Подборка | Среда | 12:00 | 🔵 Низкий |

## 🎨 Форматы постов

### Одиночный бонус
```
[КАРТИНКА]
[ВСТУПИТЕЛЬНЫЙ ТЕКСТ]

🎯 Название акции
⏰ До [дата]

[ЭМОДЗИ]

🔥 Все бонусы Спортса: [ссылка]

[КНОПКА: "Перейти к акции"]
```

### Подборка из 3 бонусов
```
[ОБЩАЯ КАРТИНКА]
[ВСТУПИТЕЛЬНЫЙ ТЕКСТ]

🎯 Акция 1 | ⏰ [дата]
🎯 Акция 2 | ⏰ [дата]
🎯 Акция 3 | ⏰ [дата]

[ЭМОДЗИ]

🔥 Все бонусы Спортса: [ссылка]

[3 КНОПКИ]
```
