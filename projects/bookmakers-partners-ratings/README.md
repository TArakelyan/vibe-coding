# Рейтинг партнерств букмекеров (bookmakers-partners-ratings)

## Описание

Интерактивная таблица партнерских соглашений российских букмекеров с клубами, федерациями и турнирами: ширина контейнера **660px**, нативные фильтры и сортировки, единый шрифт Roboto/Neoris, акцент мятный. Логотипы в таблице ведут на [Sports.ru](https://www.sports.ru/?utm_source=special-bookmakers-partners-ratings).

## Возможности

- Поиск по названию партнера, виду спорта и букмекеру.
- Три фильтра: букмекер, вид спорта, **организация** (выпадающий список с группами: клуб, федерация, турнир).
- Сортировка по колонкам: букмекер (логотип), партнер, вид спорта, год окончания сотрудничества.
- Сетка: шире первая колонка (заголовок «Букмекер» в одну строку), колонка партнера уже; заголовки и текст в ячейках **14px**, содержимое строк выровнено по вертикали по центру.
- Колонка букмекера: только логотип бренда.
- Колонка партнера: логотип организации и название; для **титульных** партнерств — бейдж «Титульный», усиленная типографика, легкий золотисто-мятный фон строки.
- Год: число, «—» если не указан, «?» если дата не определена.
- Файл `resize.js` для высоты iframe на площадке.

## Данные

- `data/bookmakers.js` — идентификаторы букмекеров и URL логотипов брендов.
- `data/partners.js` — строки таблицы: `bookmakerId`, `partnerName`, `sport`, `partnerLogo`, `endYear`, опционально `yearUnknown`, `isTitular`.
