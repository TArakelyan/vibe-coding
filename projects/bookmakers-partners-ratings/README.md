# Рейтинг партнёрств букмекеров (bookmakers-partners-ratings)

## Описание

Интерактивная таблица партнёрских соглашений российских букмекеров с клубами, федерациями и турнирами: ширина контейнера **660px**, нативные фильтры и сортировки, стиль Sports (шрифты Roboto / Sports, акцент мятный). Логотипы в таблице ведут на [Sports.ru](https://www.sports.ru/?utm_source=special-bookmakers-partners-ratings).

## Возможности

- Поиск по названию партнёра, виду спорта и букмекеру.
- Три фильтра-выпадающих списка: букмекер, вид спорта, клуб / федерация / турнир.
- Сортировка по колонкам: букмекер (логотип), партнёр, вид спорта, год окончания сотрудничества.
- Колонка букмекера: **только логотип** (без подписи названия).
- Колонка партнёра: логотип организации и название; для **титульных** партнёрств — акцентное оформление (бейдж «Титульный», усиленная типографика, лёгкий золотисто-мятный фон строки).
- Год: число, «—» если не указан, «?» если дата окончания не определена.
- Файл `resize.js` для высоты iframe на площадке.

## Данные

- `data/bookmakers.js` — идентификаторы букмекеров и URL логотипов брендов.
- `data/partners.js` — строки таблицы: `bookmakerId`, `partnerName`, `sport`, `partnerLogo`, `endYear`, опционально `yearUnknown`, `isTitular`.
