# 📊 Финансовые данные букмекеров

Современная веб-платформа для анализа финансовых данных российских букмекеров в стиле Forbes - элегантный, минималистичный дизайн с адаптивными карточками.

## ✨ Особенности

- 🎯 **Дизайн в стиле Forbes** - чистые карточки с строгой типографикой
- 📅 **Переключатель годов** - удобные табы для быстрого выбора года (2020-2024)
- 🏢 **Детальная аналитика** - полная финансовая отчетность компаний
- 💰 **Финансовые показатели** - выручка, прибыль, GGR, доля рынка, отчисления
- 📱 **Полностью адаптивный** - идеально выглядит на всех устройствах
- ⚡ **Быстрая работа** - без фреймворков, чистый JavaScript
- 🎨 **Современный UI** - карточная система с hover-эффектами

## 🎨 Дизайн-система

### Цветовая схема
- **Основной фон**: `#f5f5f5`
- **Карточки**: `#fff`
- **Границы**: `#e0e0e0`
- **Черный**: `#000`
- **Текст**: `#666` / `#999`

### Стиль Forbes
- Прямоугольные карточки без скругления
- Строгая типографика
- Минимальные тени
- Черно-белая палитра
- Четкие границы

### Компоненты
- **Переключатель годов** - табы в верхней части страницы
- **Карточки компаний** - структурированные блоки с информацией
- **Метрики** - строки с данными и индикаторами изменений
- **Логотипы** - круглые изображения компаний

## 🚀 Быстрый старт

1. Откройте `index.html` в браузере
2. Выберите год в верхней панели (2020-2024)
3. Просматривайте карточки букмекеров с финансовыми данными
4. Наводите на карточки для hover-эффектов

## 📋 Отображение данных

### Правила показа метрик
1. **Выручка** - всегда отображается
2. **Прибыль** - всегда отображается (включая отрицательные значения)
3. **GGR** - только если > 0
4. **Доля рынка** - только если > 0
5. **Целевые отчисления** - только если >= 120 млн

### Сортировка
- Компании сортируются по выручке в выбранном году (от большей к меньшей)
- Компании без данных за выбранный год не отображаются
