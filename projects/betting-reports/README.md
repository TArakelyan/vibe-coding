# 📊 Финансовые данные букмекеров

Современная веб-платформа для анализа финансовых данных российских букмекеров с простым и чистым дизайном в стиле widget-ambassadors.

## ✨ Особенности

- 🎯 **Минималистичный дизайн** - чистый серый интерфейс (#f6f6f6)
- 📅 **Фильтрация по годам** - данные за 2020-2024 годы
- 🏢 **Детальная аналитика** - полная финансовая отчетность компаний
- 💰 **Финансовые показатели** - выручка, прибыль, GGR, доля рынка, отчисления
- 📱 **Адаптивный дизайн** - работает на всех устройствах
- ⚡ **Быстрая работа** - без фреймворков, чистый JavaScript
- 🌓 **Темная тема** - автоматическое определение системных настроек

## 🎨 Дизайн-система

### Цветовая схема
- **Основной фон**: `#fff`

- **Границы**: `#d1d1d6`
- **Текст**: `#1d1d1f` / `#6e6e73`

### Компоненты
- Простые серые карточки

- Четкая типографика
- Аккуратные тени
- Плавные переходы

## 🚀 Быстрый старт

1. Откройте `index.html` в браузере
2. Выберите годы для анализа (2020-2024)
3. Изучите финансовые данные в карточках компаний
4. Сравните показатели разных лет
