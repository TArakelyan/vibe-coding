# Решение проблемы копирования виджета

## Проблема
При копировании `index.html` не подтягиваются внешние файлы (CSS, JS, данные), что приводит к нерабочему виджету.

## Решение
Создан файл **`standalone.html`** - самодостаточная версия виджета, где все стили, скрипты и данные встроены прямо в HTML.

## Что включено в standalone.html:
- ✅ Все CSS стили (включая импорты шрифтов)
- ✅ Все JavaScript код (основной скрипт + resize)
- ✅ Данные команд (teamsData)
- ✅ Данные матчей (matchesData)
- ✅ Полная функциональность (стратегии, суммы, расчеты)
- ✅ Адаптивность и мобильная версия
- ✅ Кнопка WINLINE с правильной ссылкой
- ✅ Плашка с предупреждением
- ✅ Автоматический ресайз для iframe

## Использование:
1. Скопируйте содержимое файла `standalone.html`
2. Вставьте в любое место - виджет будет работать полностью автономно
3. Никаких дополнительных файлов не требуется

## Отличия от index.html:
- Все внешние зависимости встроены внутрь
- Один файл вместо 7 отдельных (HTML + CSS + 4 JS + данные)
- Полная совместимость функционала
- Готов для встраивания в iframe или копирования

Теперь при копировании HTML-кода виджет будет работать сразу, без необходимости подключения внешних файлов.
